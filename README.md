# AccessNow+

Accessible navigation platform for discovering **barrier-free routes and accessible points of interest (POIs)**.

The project combines **interactive maps, route planning, and user features** to support more inclusive mobility in urban environments.

## Features

* Interactive map with accessible POIs  
* Accessible route planning  
* User authentication  
* Favorites system  
* REST API architecture  
* Multilingual support

## About

AccessNow+ aims to improve **accessible mobility and inclusive navigation** by combining open data, mapping technologies, and user-centered design.
